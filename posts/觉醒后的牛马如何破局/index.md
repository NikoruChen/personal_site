# 觉醒后的牛马如何破局

之前的笔记里复盘了我作为一个普通牛马，如何一步步觉醒，最终看透了上班就是拿钱吃💩，所谓的career growth不过是资本家为了引诱牛马们吃下更多的💩所创造出来的概念。那么问题来了，作为一个已经觉醒的牛马，如何才能摆脱吃💩的困局呢？毕竟大多是人还是需要钱的。
我最近一直在思考这个问题，总结下来大体思路就两条（如果有别的欢迎补充） -
第一条思路就是大家常说的”把热爱变成事业“，换言之就是找到一条不吃💩还能赚钱的路径。可以是一份有趣的、自己内心真正认可的工作；也可以是创业或自由职业，比如做自媒体、自己接项目等等。
第二条思路就是FIRE的思路，先在一段时间内硬着头皮把💩吃了，把拿到的钱攒着，攒到一定程度，被动收入可以覆盖生活开销时，就不需要再额外赚钱，也就不需要再继续吃💩了。
至于现在很流行的裸辞gap year重启人生一条龙操作，大致也是这两种思路的结合。先攒够一定存款可以支撑自己一段时间不工作，然后再在这段时间探索出一条不吃💩的赚钱路径。
这两种思路相比，我觉得FIRE更适合我，原因如下 - 
首先，我非常怀疑“不吃💩还能拿钱”这件事的可行性。赚钱的本质是为他人提供价值，这就意味着，如果我想通过自己喜欢的事赚钱，我就会在我喜欢的事里变成一个乙方的角色。做乙方不可能不妥协，妥协其实就是吃💩。而且我这个人有点“爱好洁癖”，如果在我热爱的、非常在乎事情上不能坚持己见，只会让我更痛苦，也就是说，这💩会变得更难吃。回想我当初的职业选择，其实也是因为大学时确实很喜欢编程，喜欢那种创造的感觉，才选了CS这个专业，成了码农。事实证明一旦喜欢的事成了谋生工具，吃💩就在所难免。这种吃💩的感觉又会对我的爱好形成一种反噬，导致我现在都不那么爱写代码了。
另一方面，其实真正能让你赚钱的不是你热爱的事，而是你擅长且又正好有市场需求的事，说白了就是要有product-market fit. 如果这个有PMF的事正好也是你的热爱，那恭喜你，可以在自己热爱的事里赚到钱。但大多数人没有那么幸运。大多数东亚小孩从小被放在整齐划一的模具里，从未有机会探索自己的热爱，更不要说在热爱中精进，将之培养为自己的擅长（我会说其实我的梦想是成为一名舞蹈家吗）。就算能给自己空间去探索、去精进，也未必能找到市场，实现变现，梵高的故事就是最好的例子。
总而言之，对我个人而言，“把热爱当事业”看起来是两全其美，实际操作起来可能两头都没沾上。所以我更愿意把这两件事分开，先靠擅长的事搞钱（unfortunately现阶段当码农依然是我最高效的搞钱方式），等退休了，就可以纯自嗨式地享受自己的热爱，无需迎合市场，只需取悦自己。
当然FIRE也不是适合所有人的，而且FIRE的方式也有一些弊端，这其中我最大的顾虑就是之前有篇笔记中提到过的少年游的问题。FIRE的底层逻辑是延迟满足，可人生中有很多事就是要趁年轻力壮去实现的。如果一味地追求攒钱而没能好好享受青春年华，那也是得不偿失（关于这点推荐大家去看Die with Zero这本书，里面提到的关于maximize fulfillment, memory dividend这些概念给了我很多启发，以后可以单独出篇笔记分享）。
综合以上所有的考量，我目前的计划是：短期（一到两年内）给自己gap半年到一年的时间，之后再工作几年，攒到目标资金后就回国退休。gap的好处是我能在这个年纪提前体验一下退休生活，先实现一部分的退休理想。另外我也想提前预演一下，脱离职场后会有哪些不便（目前想到的包括签证、信用卡、保险这些），这样正式退休前可以做更充分的准备。之所以选择回国（或是去其他生活成本低的国家）而不是在美国退休是因为回国所需的退休资金会比在美国少很多，这样可以少吃几年💩，趁年轻早些去享受生活。
